Home Hub — Smart Home Control Architecture

Unified control plane for Home Assistant · Apple AirPlay · Robot Vacuum · Smart Lights

Control Paths

  • • Voice/CLI → Hermes Agent → home-hub skill
  • • HA devices: HA Toolset → REST API → Docker
  • • Apple devices: pyatv → mDNS → RAOP stream
  • • TTS broadcast: macOS say → ffmpeg → RAOP

Key Dependencies

  • • macOS (pyatv, mDNS for AirPlay)
  • • Home Assistant (Docker or bare metal)
  • • pyatv · ffmpeg · Bash 4+
  • • Hermes Agent (optional — hub-ctl standalone)

Known Constraints

  • • Docker can't discover mDNS
  • • HA tokens expire every 30 min
  • • RAOP needs 3× audio gain
  • • Apple TV (tvOS 18) pairing blocked
